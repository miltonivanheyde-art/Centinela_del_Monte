# PITCH_CANONICAL_LANGUAGE

## Objetivo
Establecer el lenguaje oficial de la capa investor-ready para mantener coherencia narrativa, técnica y doctrinal.

## Definiciones Obligatorias
- Centinela no se describe como "IoT ganadero".
- Centinela se describe como infraestructura territorial resiliente.
- Centinela se describe como sistema de persistencia operativa.
- Centinela se describe como mecanismo para reducción de incertidumbre rural.
- Centinela se enmarca en soberanía tecnológica aplicada a operación territorial.

## Terminología Principal
- infraestructura territorial resiliente
- persistencia operativa
- reducción de incertidumbre rural
- soberanía tecnológica

## Tono Editorial Obligatorio
- técnico
- sobrio
- no marketinero
- sin claims no verificables

## Reglas de Redacción
- Priorizar lenguaje operacional y verificable.
- Evitar promesas de performance no medidas.
- Evitar superlativos comerciales sin respaldo técnico.
- Mantener separación entre narrativa inversor (`docs/pitch/`) y core técnico (`main/`, documentación doctrinal base).

# Investor Pitch

Este directorio contiene todos los artefactos investor-ready
del Proyecto Centinela del Monte.

Objetivo:
transformar la arquitectura técnica y doctrinal
en una presentación clara, demostrable e invertible.

## Propósito de la Carpeta
- Consolidar narrativa ejecutiva para inversores sin alterar el core técnico.
- Ordenar artefactos de presentación, riesgo, negocio y demo en una base gobernada.

## Flujo Recomendado
1. Validar estado base en `BASELINE_STATUS_v1.md`.
2. Revisar narrativa estratégica en `executive_summary/`.
3. Recorrer propuesta visual en `investor_deck/` y `visuals/`.
4. Contrastar viabilidad con `business/`, `roadmap/` y `risks/`.
5. Cerrar con enfoque demostrable en `demo/`.

## Orden de Lectura para Inversores
1. `BASELINE_STATUS_v1.md`
2. `executive_summary/EXECUTIVE_SUMMARY_v1.md`
3. `investor_deck/INVESTOR_DECK_v1.md`
4. `architecture/SYSTEM_OVERVIEW_v1.md`
5. `business/BUSINESS_MODEL_v1.md`
6. `roadmap/ROADMAP_v1.md`
7. `risks/RISK_REGISTER_v1.md`
8. `demo/DEMO_SCRIPT_v1.md`

## Separación de Capas
- Core técnico: permanece en `main/`, `docs/` doctrinal y configuración ESP-IDF.
- Narrativa inversor: se desarrolla exclusivamente en `docs/pitch/`.

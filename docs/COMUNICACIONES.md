## Especificaciones Detalladas de Comunicaciones

## 1. Módulo de Radio (Ref: ADR-HEARTBEAT)

- **Tecnología:** LoRaWAN (preferido) o Módulo Satelital de bajo consumo.
- **Frecuencia:** Banda ISM (ej. 868 MHz o 915 MHz para LoRaWAN).
- **Potencia de Transmisión:** Ajustable, hasta +20 dBm.
- **Sensibilidad de Recepción:** < -130 dBm.

## 2. Antena

- **Tipo:** Externa, omnidireccional.
- **Ganancia:** Mínimo 3 dBi.
- **Protección:** IP67, resistente a UV y condiciones climáticas extremas.
- **Conector:** SMA o N-Type.

# Especificaciones Detalladas de Sensores

## 1. Sensor de Temperatura y Humedad
- **Tipo:** Digital (ej. SHT31, BME280).
- **Interfaz:** I2C.
- **Rango de Temperatura:** -40°C a +85°C.
- **Precisión de Temperatura:** ±0.5°C.

## 2. Sensor de Movimiento
- **Tipo:** Acelerómetro de 3 ejes (ej. ADXL345).
- **Interfaz:** I2C o SPI.
- **Rango de Detección:** ±2g a ±16g.

## 3. Sensor de Integridad (Tamper Switch)
- **Tipo:** Switch mecánico de contacto.
